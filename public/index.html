三週期末生存挑戰

本網站結合個人網站、像素風場景與視覺小說玩法。玩家要在 21 天內安排每天四個行動時段，在體力、壓力、金錢、知識與開心值之間做取捨，完成期末專題及學業要求。

遊戲包含劇情選項、場景切換、數值成長、超商道具、隨機事件、時間濾鏡、自動存檔、結局判定與 MongoDB 彈幕資料庫，讓課程中的前端互動、API 與 CRUD 實作整合成完整作品。

前端HTML / CSS / JavaScript 負責畫面、遊戲邏輯、動畫與音效。
後端Node.js + Express 提供存檔與彈幕 API。
資料庫MongoDB 儲存玩家存檔、彈幕與可延伸資料。
部署GitHub 管理版本，Render 部署線上網站。
